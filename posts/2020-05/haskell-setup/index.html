安装 Haskell Stack

Haskell Stack 1 是一个 Haskell 的工具链管理器，在 Windows 平台上可以通过 scoop 下载:

scoop install stack

在 scoop info haskell 时，也会给出提示，将用户引导至 stack 工具。

安装好 stack 后，配置 THU（清华大学） 的镜像 2，不过 Windows 系统上的 stack 配置文件不在 ~/.stack/config.yaml，而是 %APPDATA%\stack\config.yaml。

然后，通过 stack setup 安装本机适应的工具链。

\begin{equation*} \begin{aligned} \text{GHC} & \approx 200 \mathrm{MB} \\ \text{MSYS2} & \approx 60 \mathrm{MB} \\ \end{aligned} \end{equation*}

之后，安装 Haskell 的包管理器 Cabal 3 ， 并配置镜像 2。
